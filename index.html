Modules

Everything you need.

⚡
Hook System
datalib:systems/hook/

Event-driven callbacks for block breaks, placements, kills, dimension changes, item use, fishing and more — advancement-backed, zero tick overhead.

⏱
Rate Limit
datalib:systems/rate_limit/

Per-player cooldown enforcement via scoreboard. Prevent spam, throttle events, gate abilities — configurable per call site.

🔢
Math Handler
datalib:systems/math/

Integer and fixed-point arithmetic utilities. Multiply, divide, clamp, abs, min/max — without floating-point hacks.

📦
NBT Handler
datalib:systems/nbt/

Storage-backed NBT read/write helpers, deep path manipulation, typed getters, merge utilities.

🔤
String Library
stringlib:

Macro-powered string processing. Substring, concat, length, split, case conversion — embedded as a full overlay namespace.

🖥
Dev Settings GUI
datalib:systems/dev_settings/

Written book GUI with scoreboard navigation and storage-backed state. Accessible via #datalib:admin/menu.

🔗
Load Integration
dl_load: / load:

RunLoad (LanternLoad fork) embedded. Self-version score, #load:reload tag support, macro-based load:api/require.

🎮
PlayerAction
player_action:

v1.7 embedded dependency. Click entity events, ride detection predicates, interaction advancement hooks.
